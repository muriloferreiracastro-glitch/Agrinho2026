# 🌱 Agrinho 2026 - Sustentabilidade

Plataforma educacional dedicada à sustentabilidade ambiental e responsabilidade ecológica.

## 📋 Descrição

O Agrinho 2026 é um site moderno e responsivo que promove:
- Educação ambiental
- Práticas sustentáveis
- Consciência ecológica
- Ações para um planeta melhor

## ✨ Características

- ✅ **Design Responsivo**: Funciona perfeitamente em desktop, tablet e mobile
- ✅ **Interface Moderna**: Gradientes, animações suaves e interatividade
- ✅ **Acessível**: Semântica HTML5 correta
- ✅ **Performance**: Código otimizado e leve
- ✅ **SEO Friendly**: Meta tags e estrutura apropriada
- ✅ **Formulário de Contato**: Para interação com usuários

## 🚀 Estrutura do Projeto

```
Agrinho2026/
├── index.html           # Página principal
├── styles/
│   ├── main.css        # Estilos principais
│   └── responsive.css  # Media queries
├── js/
│   └── main.js         # Lógica JavaScript
└── README.md           # Este arquivo
```

## 🎨 Temas Principais

1. **Educação Ambiental** - Conscientização sobre problemas ambientais
2. **Reciclagem e Reuso** - Redução de desperdício
3. **Agricultura Sustentável** - Práticas agrícolas responsáveis
4. **Energias Renováveis** - Fontes limpas de energia
5. **Conservação de Água** - Proteção de recursos hídricos
6. **Reflorestamento** - Restauração de florestas

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica
- **CSS3**: Gradientes, animações, Flexbox e Grid
- **JavaScript (Vanilla)**: Sem dependências
- **Responsive Design**: Mobile-first

## 📱 Breakpoints Responsivos

- **Desktop**: > 768px
- **Tablet**: 481px - 768px
- **Mobile**: ≤ 480px

## 🏃 Como Usar

1. Clone o repositório
2. Abra `index.html` em seu navegador
3. Navegue pelos temas de sustentabilidade
4. Use o formulário de contato para enviar mensagens

## 🔍 Funcionalidades JavaScript

- Carregamento dinâmico de cartões de temas
- Manipulação de formulário com validação
- Intersection Observer para animações ao scroll
- Navegação suave entre seções
